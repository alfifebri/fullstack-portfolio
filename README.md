# Professional React Portfolio

A modern, responsive personal portfolio website built with React and Vite. Designed to showcase skills and projects with a clean, professional aesthetic.

## Features

- **Modern UI/UX**: Premium dark theme with smooth gradients and animations.
- **Responsive Design**: Fully responsive layout for mobile, tablet, and desktop.
- **Component-Based**: Modular structure using React components.
- **Interactive**: Smooth scrolling logic and hover effects.
- **Contact Form**: Frontend implementation ready for integration.

## Tech Stack

- **React**: UI library
- **Vite**: Next Generation Frontend Tooling
- **Lucide React**: Beautiful & consistent icons
- **CSS3**: Custom properties and flexbox/grid layouts

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/alfifebri/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment being GitHub Pages

1. Update `vite.config.js` if necessary (base path is set to `./`).
2. Run the build command:
   ```bash
   npm run build
   ```
3. Deploy the `dist` folder to GitHub Pages.

## Project Structure

```
src/
├── components/     # Reusable UI components (Navbar, Hero, etc.)
├── pages/          # Page layouts (Home.jsx)
├── assets/         # Static assets
└── index.css       # Global styles and variables
```

## Customization

- **Colors**: Edit the CSS variables in `src/index.css`.
- **Content**: Update the data arrays in `src/components/` files to reflect your own information.
